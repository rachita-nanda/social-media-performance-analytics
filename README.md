Social Media Performance Analytics & Growth Optimization
Project Overview

This freelance analytics project focuses on evaluating social media campaign performance and optimizing growth strategies using data-driven insights. The objective was to analyze engagement, revenue trends, customer behavior, and campaign effectiveness to support strategic decision-making.

Business Objectives

Measure campaign ROI and revenue contribution

Perform RFM segmentation for customer value analysis

Track Month-over-Month (MoM) and Year-to-Date (YTD) growth

Identify high-performing customer segments

Optimize marketing spend allocation

Tools & Technologies Used

SQL – Data extraction & transformation

Python – Exploratory Data Analysis & insights

Power BI – Interactive KPI dashboards

Excel – RFM modeling & performance metrics

Key Deliverables

Campaign performance dashboard

Customer segmentation using RFM model

Revenue growth trend analysis

Strategic recommendations for growth optimization

This project demonstrates end-to-end analytics workflow from raw data to business insights and dashboard reporting.
